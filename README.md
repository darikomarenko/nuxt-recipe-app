# Nuxt App using API

Stack: Nuxt3 ( + Nuxt 4 features), TypeScript, Tailwind, Nuxt Modules: Image, Icon, GoogleFonts
<br/>
See project deployment at: https://nuxt-recipe-app77.vercel.app

## Setup

npm install

## Development Server

Start the development server on `http://localhost:3000`:

npm run dev
